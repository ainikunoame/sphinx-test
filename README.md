## memo

各自上位ディレクトリにsphinx仮想環境作成

```
py -m venv \sphinx-venv
```
仮想環境をアクティベート
```
\sphinx-venv\Scripts\activate
```
sphinxのインストール
```
pip install sphinx Pillow
```
インストールの確認
```
sphinx-quickstart --version
```
